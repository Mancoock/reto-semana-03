# Reto Semana 3: Analizador de Ventas 📊
**Programación para Ciencia de Datos**

Este proyecto procesa un archivo CSV con transacciones de ventas, agrupa los datos por producto y calcula métricas clave:
* Unidades vendidas
* Ingreso total
* Precio promedio

El resultado final es un nuevo reporte en formato CSV, ordenado de mayor a menor ingreso total, implementando el uso de diccionarios en Python.
